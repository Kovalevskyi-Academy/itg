## UP and DOWN a service in containers.
### REQUIREMENTS
- docker & docker-compose
- Java 17
- MAVEN 3.8.3++
- GNU Make 4.2.1++

### UP & DOWN
1. Run: `make -f textify.mk up`.
   Service be alive even after host restart!
2. Run: `make -f textify.mk down`.
